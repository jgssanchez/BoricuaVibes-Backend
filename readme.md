## 🔧 Instalación

Clona el repositorio de Boricua Vibes Backend:
```bash
git clone https://github.com/jgssanchez/BoricuaVibes-Backend.git

Instala las dependencias:

bash:
npm install

🤖 Uso
Para iniciar el servidor de desarrollo, ejecuta:

bash:

npm run dev

```
